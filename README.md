# Protótipo de Chatbot – Pesquisa

Este é um protótipo simples de chatbot em Flask, pensado para testes exploratórios em pesquisas culturais, históricas e simbólicas.

## Como rodar
```bash
pip install -r requirements.txt
python main.py
```

Acesse: http://localhost:3000

## Observação
As respostas são baseadas em palavras-chave e podem ser adaptadas para entrevistas, coleta de dados ou simulações conversacionais.
